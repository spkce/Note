dnn(Deep Neural Networks, 深度神经网络). 作用是载入别的深度学习框架(TensorFlow、Caffe、Torch)中已训练好的模型，然后完成该模型的预测等工作

1. dnn加载Caffe模型
```c++
cv::dnn::Net net = cv::dnn::readNetFromCaffe("xxx.prototxt", "xxx.caffemodel");
```
xxx.prototxt是Caffe框架的网络结构定义文件，描述神经网络各层的连接方式和参数配置
xxx.caffemodel是Caffe训练生成的二进制权重文件，包含网络各层的参数值。


使用Caffe模型做人脸检测
```c++

    bool 	crop = false, //crop 参数设置为 true，则在调整图像大小时进行中心裁剪，以确保输出尺寸与指定的尺寸精确匹配
    int 	ddepth = CV_32F 
)
*/
Mat blob = blobFromImage(frame, 1.0, Size(300, 300), Scalar(104, 177, 123)); 
detector.setInput(blob);
Mat detections = detector.forward();
/*
detections是一个4D张量，其维度结构为：

第1维(0)：固定为0（批处理维度，单图像时为0）
第2维(0)：固定为0（检测结果索引）
第3维(i)：第i个检测框的索引

*/
for (int i = 0; i < detections.size[2]; ++i) //第3维度有每一个检测结果
```
