# 着色器版本规范
在顶点着色器和片段着色器第1行总是声明着色器版本
```c++
#version 300 es
```

## 变量

### 标量
float 、int、uint、bool

### 浮点向量
float、 vec2、vec3、vec4

### 整数向量
int 、ivec2、ivec3、ivec4

### 无符号整数向量
uint 、uvec2、uvec3、uvec4

### 布尔向量
bool 、bvec2、bvec3、bvec4


### 矩阵
mat2(mat2x2)、mat2x3、mat2x4
mat3x2、mat3(mat3x3)、mat3x4
mat4x2、mat4x3、mat4(mat4x4)

### 变量构造
```c++
vec4 myVec4 = vec4(1.0) // myVec4={1.0,1.0,1.0,1.0}
mat3 mayMat3 = mat3(1.0,0.0,0.0 //first column
                    0.0,1.0,0.0 //second column
                    0.0,1.0,1.0 //third colum
            )
```

## 输入输出限定符
着色器输入变量用in修饰，表示由应用程序指定值
顶点着色器输出变量用out修饰，它会传给片段作色器。顶点着色器的out在片段作色器中会以in修饰
```
#version 300 es
uniform mat4 u_matViewProjection;
layout(location = 0) in vec4 a_position;
layout(location = 1) in vec4 a_color;
out vec3 v_color;
void main(void)
{
    gl_Position = u_matViewProjection * a_position;
    v_color = a_color;
}
```
```
#version 300 es
in vec3 v_color;
layout(location = 0) out vec4 o_fragClolor;
void main(void)
{
    o_fragClolor = vec4(v_color, 1.0);
}
```
## 插值限定符
顶点着色器输出、片段着色器中输入,默认是平滑着色。
显式声明平滑着色
```
// vertex shader
smooth out vec3 v_color
// fragment shader
smooth out vec3 v_color
```
显式声明平面着色
```
// vertex shader
flat out vec3 v_color
// fragment shader
flat out vec3 v_color
```
## 精度限定符
lowp（10位定点表示）、mediump（16位浮点）、highp（32位浮点表示）
默认精度限定符 precision 

# varying
在GLSL（OpenGL着色器语言）中，‌varying‌指在顶点着色器中声明的变量，其值需要在后续的片元着色器处理中被插值传递。这类变量通常用于表示需要跨渲染管线传递的顶点数据（如坐标、法线等），但在片元着色器中不可直接声明或修改。

# 插值
插值是指在光栅化阶段对顶点属性进行平滑过渡计算的过程，使得顶点之间的像素能获得连续变化的属性值
