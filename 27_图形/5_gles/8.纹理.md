纹理（Texture）‌ 是用于存储图像数据（如颜色、深度、法线等）的核心对象，主要用于 ‌贴图（Mapping）‌ 和 ‌渲染管线‌ 中的采样（Sampling）。

### 使用精灵图作为纹理

```c++
// 1. 生成纹理
GLuint textureId;
glGenTextures(1, &textureId);
glBindTexture(GL_TEXTURE_2D, textureId);

// 2. 设置纹理参数
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MIN_FILTER, GL_LINEAR);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MAG_FILTER, GL_LINEAR);

// 3. 加载图像数据（如从文件读取）
glTexImage2D(
    GL_TEXTURE_2D, 0,         // 层级
    GL_RGBA,                  // 格式
    width, height,            // 尺寸
    0,                        // 边框
    GL_RGBA,                  // 格式
    GL_UNSIGNED_BYTE,         // 数据类型
    imageData                 // 像素数据
);
```
精灵图通常包含多个子图（如动画帧），需通过UV坐标定位
```c++
// 示例：从精灵图中提取第3行第2列的帧（假设每帧64x64）
float frameX = 2; // 列索引（从0开始）
float frameY = 2; // 行索引
float frameWidth = 64.0f / spriteSheetWidth;
float frameHeight = 64.0f / spriteSheetHeight;

// 计算UV坐标（左下角为原点）
float uvLeft   = frameX * frameWidth;
float uvRight  = (frameX + 1) * frameWidth;
float uvBottom = 1.0f - (frameY + 1) * frameHeight; // OpenGL纹理坐标Y轴反向
float uvTop    = 1.0f - frameY * frameHeight;
```
着色器代码：
```glsl
// 顶点着色器
attribute vec2 aTexCoord;
varying vec2 vTexCoord;
void main() {
    vTexCoord = aTexCoord; // 传递UV坐标
    // ...其他变换
}

// 片段着色器
uniform sampler2D uSpriteSheet;
varying vec2 vTexCoord;
void main() {
    gl_FragColor = texture2D(uSpriteSheet, vTexCoord);
}
```

### 采样器Sampler
在OpenGL ES中，‌采样器（Sampler）‌是着色器中用于访问纹理对象的特殊变量类型，它定义了如何从纹理中读取数据（即采样规则）

采样器绑定到纹理单元后，通过纹理坐标（UV）从纹理中提取颜色值
```glsl
.....
uniform sampler2D uTexture;  // 2D纹理采样器
void main() {
    gl_FragColor = texture2D(uSpriteSheet, vTexCoord);
}
```
